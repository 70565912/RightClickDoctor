# Third Party Notices

RightClickDoctor currently has no packaged third-party runtime dependencies.

The implementation was designed after reviewing public documentation and similar open-source projects:

- `moudey/Shell` for modern context-menu customization concepts.
- `oleg-shilo/shell-x` for Shell extension registration patterns.
- `yanxijian/ShellExtContextMenuHandler` for a compact context-menu handler example.

No source code from those repositories is vendored in this project.
